<h1>OX Mediboard</h1>

<h2>About</h2>

Mediboard is an open source health facility management application.

<h2>Requirements</h2>
<ul>
    <li>Apache</li>
    <li>Php</li>
    <li>Mysql</li>
    <li>Pecl</li>
    <li>Composer</li>
    <li>Git</li>
</ul>

<h2>Installation</h2>
```
git clone project
composer install
npm run build-vue-dev
composer ox-install-config
composer ox-install-database
 ```
